4 in Line

  • January 2024
    • •
  • GitHub Game Link,

Project Description:

This game was developed in Kotlin as the final project of the subject (Fundamentos de Programação)

This game resembles the game "Four-in-line", where you play against the computer. It has the same goal, to align four balloons in a row (vertically, horizontally, or diagonally), but the balloons go up instead of going down.

There is another functionality called "explodir balao" where it removes a balloon from the computer and vice versa. This is a simple game that you can play through the command line.
